1.- Introducción

SAFET (Sistema Automatizado de Firma Electrónica y Estampillado de Tiempo) es una herramienta basada totalmente en software libre (de libre uso y distribución), multiplaforma, que permite a personas que interactúan con sistemas de información, gestionar documentos electrónicos, verificar, modelar y visualizar de forma inteligible, inteligente, personalizada y adaptada sus datos y activos de información, utilizando diagramas, gráficos, tablas, listas, reportes, o salidas personalizadas apreciables en Internet o en aplicaciones de escritorio, de forma tal que, se cuente como elemento adicional, la confianza de la identificación legal en Internet que es la firma electrónica. 4.- Funcionalidades de la aplicación

A continuación se presenta las funcionalidades de SAFET:

  • Gestionar flujos de trabajos
  • Implementación de firma electrónica y estampillado de tiempo
  • Interacción con sistema trac
  • Visualización de los flujos de trabajos
  • Modelar sistemas de información
  • Gestionar documentos electrónicos

5.- Arquitectura de la aplicación



Figura S1-1 Arquitectura de SAFET


  • Digidoc: Proyecto del Gobierno de Estonia, que implementa el estándar openXADES (XML + firma electrónica), es compatible con tarjetas inteligentes a través de opencs, utiliza el esquema pki.
  • Graphviz: Herramienta para la visualización de gráficos.
  • Qt: Librería profesional multiplataforma de programación (Trolltech Qt)
  • libsafet: Una librería para que desarrolladores de C++ implementen funcionalidades de gestión de flujo de trabajo, firma electrónica y estampillado de tiempo
  • inflow: Interfaz Gráfica para gestión de flujos de trabajo con firma electrónica (escritorio web) para usuarios y modelado
  • safet: Programa de consola para gestión de flujos de trabajo con firma para administradores y usuarios

6.- Descripción

Software para la gestión de información a través de flujos de trabajo (workflows), y firma electrónica bajo formato OpenXaDES (www.openxades.prg)
