15.- Archivo de Configuración safet.conf


Este archivo permite configurar la librería libsafet, este archivo esta compuesto por varias secciones entre las que se encuentra:

  • [Log]: en esta sección se agrega la ruta absoluta donde esta el archivo se registra los eventos de la aplicación, y donde se habilita las diferentes opciones de registro (depuración, errores, advertencia) ejemplo:
    log.filepath = /home/pbuitrago/.safet/log -> dirección absoluta

  • [Widgets]: en esta sección se agrega la ruta absoluta donde se encuentra almacenado los documentos de flujo de trabajo ejemplo:
    widgets.getfilewidget.* = /home/pbuitrago/.safet/flowfiles/ticketstrac.xml
    widgets.getfilewidget.* = /home/pbuitrago/.safet/flowfiles/ticketstracavanzado.xml

  • [Database]: en esta sección se encuentra la información de configuración de base de datos relacional, entre las informaciones se encuentra:
    Puerto de conexión (database.port.1 = 5432)
    Nombre de usuario (database.user.1 = nombre_usuario)
    Contraseña del usuario (database.password.1 = xxxxxx)
