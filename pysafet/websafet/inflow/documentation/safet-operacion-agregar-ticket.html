12.3.1.- Operación:agregar_ticket:


Esta operación permite agregar un nuevo ticket. Cuando se ejecuta un click sobre el botón operacion:agregar_ticket, entonces en el área de cuadro de edición se agrega la linea de texto operacion:agregar_ticket (ver figura S3.1.1-1) y en el menú inteligente se despliega los campos relacionados con la operación seleccionada (ver figura S3.1.1-2)


figura S3.1.1-1. Área cuadro de edición selección de la operacion:agregar_ticket


Figura S3.1.1-2. Menú Inteligente, campos de la operación agregar_ticket


Para agregar un nuevo ticket se necesita que se ingrese todos los campos que aparece en el menú inteligente que se muestra en la figura 11. Aquellas acciones que aparecen con el caracter asterisco ( * ) al final de cada titulo son de entrada obligatoria. Entre las campos se encuentra:


  • resumen* en este campo se ingresa un resumen sobre el ticket, para agregar este campo debe hacer clik sobre resumen* del menú inteligente. Una vez seleccionada este campo, se debe agregar al cuadro de edición la linea de texto resumen: (ver figura S3.1.1-3) y debe desaparecer del menú inteligente el campo resumen*. (ver figura S3.1.1-4) Este campo es obligatoria agregar.

    figura S3.1.1-3. Área cuadro de edición selección del campo resumen*


    figura S3.1.1-4. Menú inteligente. No aparece el campo resumen*


    Una vez que ingresa un texto valido para el resumen del ticket se le da click al botón

    Si quiere eliminar el campo resumen: se le da click al botón


  • descripcion*: en este campo se debe ingresar una pequeña descripción del ticket, para agregar este campo se debe hacer click sobre descripción* del menú inteligente, una vez seleccionada este campo, se debe agregar al cuadro de edición la linea de texto descripción: (ver figura S3.1.1-5) y debe desaparecer del menú inteligente el campo resumen*. (ver figura S3.1.1-6) Este campo es obligatoria agregar.

    Figura S3.1.1-5. Área cuadro de edición selección del campo descripción*


    figura S3.1.1-6. Menú inteligente. No aparece el campo descripción*


    Una vez que ingresa el descripción del ticket se le da click al botón
    si quiere eliminar la campo descripcion* se le da click al botón



  • componente* para este campo se debe seleccionar una de las opciones de los componentes que aparecen del ticket, para agregar esta campo se debe hacer click sobre componente* del menú inteligente, una vez seleccionada este campo, se debe agregar al cuadro de edición la linea de texto componente: y un botón de selección (ver figura S3.1.1-7) y debe desaparecer del menú inteligente la campo componente*. (ver figura S3.1.1-8) Este campo es obligatoria agregar.

    Figura S3.1.1-7. Área cuadro de edición selección de la opción componente*, botón de selección.


    Nota importante: Si al pulsar el botón de selección no aparece ninguna opción, es probable que existe un error de conexión a la base de datos. En este caso se recomienda revisar archivo safet.log para tener mayor información sobre el posible error.

    Figura S3.1.1-8. Menú inteligente. No aparece el campo componente*


    Una vez que selecciona la opción del componente del ticket se le da click al botón
    Si quiere eliminar la campo componente* se le da click al botón


  • hito*: en este campo se debe seleccionar una de las opciones que aparecen de los hitos del ticket, para agregar este campo se debe hacer click sobre hito* del menú inteligente, una vez seleccionada este campo, se debe agregar al cuadro de edición la linea de texto hito: y un botón de selección (ver figura S3.1.1-9), debe desaparecer del menú inteligente componente*. (ver figura S3.1.1-10) Este campo es obligatoria agregar.

    Figura S3.1.1-9. Área cuadro de edición selección del campo hito*, botón de selección.


    Nota importante: Si al pulsar el botón de selección no aparece ninguna opción, es probable que existe un error de conexión a la base de datos. En este caso se recomienda revisar archivo safet.log para tener mayor información sobre el posible error.

    Si quiere eliminar el campo hito* se le da click al botón




  • version* en este campo se debe seleccionar una de las opciones que aparecen en la versión del ticket, para agregar este campo se debe hacer click sobre version* del menú inteligente, una vez seleccionada este campo, se debe agregar al cuadro de edición la linea de texto version: y un botón de selección (ver figura S3.1.1-11) debe desaparecer del menú inteligente el campo version* (ver figura S3.1.1-12). Este campo es obligatoria agregar.

(ver figura S3.1.1-14) Este campo es obligatoria agregar.

  • prioridad* en este campo se debe seleccionar una de las opciones de las prioridades que aparecen en el ticket, para agregar este campo se debe hacer click sobre priorida* del menú inteligente, una vez seleccionada este campo, se debe agregar al cuadro de edición la linea de texto priorida: y un botón de selección (ver figura S3.1.1-15) y debe desaparecer del menú inteligente el campo priorida*. (ver figura S3.1.1-16) Este campo es obligatoria agregar.
