SAFET: InFlow

Interfaz de Usuario Gráfica para el Sistema SAFET.

Aplicación de escritorio para la entrada de datos en el sistema SAFET. Se pueden gestionar flujos de trabajo con firma electrónica y estampillado de tiempo, así como aplicar firma electrónica a documentos utilizando el estándar XAdES.

El soporte para dispositivos criptográficos como tarjetas inteligente está incoporado de forma nativa en InFlow.
